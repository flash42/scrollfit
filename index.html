Help defining scroll/swipe speed.
We are collecting Your input to be able to define scroll speed. For this you only need to use your mouse scroll or touchpad and we will measure the movements and save it for later use.
How to measure?
  • Scroll once with speed in the textbox below
  • Send Data button will appear. Click it to send the collected data.
Use Reset Data if you made a mistake and want to try again without sending data.
